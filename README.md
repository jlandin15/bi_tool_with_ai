# BI Tool with AI

Este es un proyecto de visualización masiva de datos utilizando D3.js integrado con herramientas de inteligencia artificial. El objetivo es crear visualizaciones interactivas y dinámicas para análisis de datos de negocio.

## Estructura del Proyecto

- **src/assets**: Recursos estáticos como imágenes y fuentes.
- **src/js**: Scripts de JavaScript para visualizaciones con D3.js.
- **src/css**: Estilos CSS para la interfaz.
- **src/data**: Datos de entrada para las visualizaciones.
- **docs**: Documentación del proyecto.

## Estado

Este proyecto está en fase de configuración inicial. Próximos pasos incluyen la definición de requisitos específicos y el desarrollo de visualizaciones.
